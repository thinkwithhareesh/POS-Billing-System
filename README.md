# Hareesh Restaurant POS 🍽️

A modern and responsive Restaurant Point of Sale (POS) system built for managing orders, billing, menu items, and sales reporting.
This project is developed using HTML, CSS, and JavaScript without any backend or database setup.

## 📌 Features

### 🛒 POS Billing System
- Add menu items to cart
- Increase/decrease item quantity
- Automatic subtotal, tax, and total calculation
- Fast checkout experience

### 🍔 Menu Management
- Add new food items
- Update prices and images
- Dynamic menu display

### 📊 Sales Dashboard
- Track total revenue
- View total orders
- Display recent transactions

### 💳 QR Payment Support
- Dynamic QR code generation
- Easy payment scanning support

### 🧾 Printable Receipt
- Generate customer bills
- Print receipts directly from browser

### 📱 Responsive UI
- Mobile-friendly layout
- Modern glassmorphism design
- Smooth animations and transitions

## 🛠️ Tech Stack

| Technology | Purpose |

## 🚀 How to Run the Project

### ✅ Method 1 — Open Directly in Browser
1. Open the project folder
2. Double-click `index.html`

The application will open in your default browser.

### ✅ Method 2 — Using VS Code Live Server
**Steps:**
1. Open the project in Visual Studio Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Select: **Open with Live Server**

The project will start automatically in your browser.

### ✅ Method 3 — Using Python HTTP Server
1. Open Terminal in Project Folder
2. Run: `python -m http.server 8000`
3. Open Browser: `http://localhost:8000`
